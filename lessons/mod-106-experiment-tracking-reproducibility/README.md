# mod-106-experiment-tracking-reproducibility: Experiment Tracking & Reproducibility

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 10 hours

## Learning objectives

- Track experiments with MLflow (and an alternative such as W&B)
- Control randomness and pin environments for byte-comparable reruns
- Version datasets and models (DVC, MLflow Model Registry)
- Run a small experiment sweep and write a reproducibility report
