# mod-110-ml-systems-design: ML Systems Design

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 14 hours

## Learning objectives

- Walk a problem from business goal to ML system blueprint
- Identify training/serving skew, data flywheels, and feedback loops
- Design an A/B test with appropriate guardrails
- Decide build-vs-buy and when to delegate to peer specialist tracks (LLM application, RAG, fine-tuning, evaluation)

## Structure

- `01-…md` … `0N-…md`: lecture chapters.
- `exercises/`: per-exercise prompts.
- `labs/`: long-form hands-on labs.
- `quizzes/`: knowledge checks.
- `resources.md`: external references.
