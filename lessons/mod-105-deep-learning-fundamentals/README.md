# mod-105-deep-learning-fundamentals: Deep Learning Fundamentals with PyTorch

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 18 hours

## Learning objectives

- Write a PyTorch training loop from scratch (forward, loss, backward, step)
- Build an MLP, a CNN, and a small Transformer encoder
- Apply regularisation, learning-rate schedules, and mixed-precision training
- Gain working familiarity with transformer LLMs and where to delegate to specialist tracks

- `01-…md` … `0N-…md`: lecture chapters.

- `quizzes/`: knowledge checks.
