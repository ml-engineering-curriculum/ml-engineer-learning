# mod-108-model-deployment-serving: Model Packaging & Deployment

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 14 hours

## Learning objectives

- Package a trained model behind a FastAPI service with health checks and structured logs
- Containerise the service with Docker and run it locally and in CI
- Implement batch and online inference paths and reason about latency vs. throughput
- Hand off to the AI Infrastructure Engineer / ML Platform tracks for Kubernetes deployment
