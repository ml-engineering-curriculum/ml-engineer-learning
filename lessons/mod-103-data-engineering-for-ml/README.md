# mod-103-data-engineering-for-ml: Data Engineering for ML: Ingestion, Cleaning, Feature Engineering

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 14 hours

## Learning objectives

- Ingest data from files, SQL, and object stores into ML-ready datasets
- Profile and clean messy data: missingness, outliers, leakage, label quality
- Engineer features for tabular, text, and image data
- Validate datasets with schema and distribution checks (Great Expectations / pandera)

## Structure

- `01-…md` … `0N-…md`: lecture chapters.
- `exercises/`: per-exercise prompts.
- `labs/`: long-form hands-on labs.
- `quizzes/`: knowledge checks.
- `resources.md`: external references.
