# mod-104-classical-ml-modeling: Classical ML Modeling: Regression, Trees, Ensembles

> Scaffolded by `aicg org execute-plan`. Lecture chapters and exercise content are authored on subsequent autonomous cycles.

**Estimated effort:** 16 hours

## Learning objectives

- Train and tune linear/logistic regression with regularisation
- Train decision trees, random forests, gradient-boosted trees (XGBoost / LightGBM)
- Apply k-fold cross-validation and hyperparameter search (grid / random / Optuna)
- Interpret models with feature importance and SHAP and avoid common selection pitfalls
